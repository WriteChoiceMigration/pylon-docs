## Setup

Click into the App Directory from your Pylon dashboard and then click on the Discord integration to view setup instructions.

## Usage

<Frame>
<img alt="" class="block" data-testid="zoom-image" fetchpriority="high" height="502" sizes="(max-width: 640px) 400px, 768px" src="/gif/CleanShot 2025-03-12 at 16.47.53.gif" width="800"/>

1. Link the channel in Pylon by hitting the Add Account button on the Accounts Page

    <Frame>
    <img alt="" class="block" data-testid="zoom-image" fetchpriority="low" height="180" loading="lazy" sizes="(max-width: 640px) 400px, 768px" src="/images/d1de273b.png" width="1080"/>
    </Frame>

2. Select the channel(s) from the list and hit save.
3. Done!

New posts in f[orum channels](https://support.discord.com/hc/en-us/articles/6208479917079-Forum-Channels-FAQ) will automatically create issues in pylon with **complete bidirectional syncing, including messages, files, and reactions**. All Pylon features can then be used to manage these issues.

In [regular channels](https://discord.com/community/basic-channel-setup), either the channel can be set to turn every message into a ticket or users can use the ticket emoji ðŸŽ« in order to open a ticket explicitly

Discord issues can be identified by setting the Issue Source filter or visually by the Discord logo on issues.
