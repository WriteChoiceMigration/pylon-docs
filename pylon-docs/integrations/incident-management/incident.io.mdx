## Setup

Click into the App Directory from your Pylon dashboard and then click on the incident.io integration to view setup instructions.

## Usage

Within the issue view, connect customer issues with the appropriate incident using the incident.io module on the sidebar.

The Pylon issue to incident link is now available across the product:

* Create a view of tickets filtered down by a specific incident
* Run reporting & analytics on how many tickets are associated with incidents
* Create differentiated automated workflows for tickets that are tied to an incident
