Select multiple issues in the Kanban or List view to take bulk actions on them. Issues can be selected using the checkbox on the top right of the card in Kanban or on the left of the row in List view.

<Frame>
<img alt="" class="block" data-testid="zoom-image" fetchpriority="high" height="181" sizes="(max-width: 640px) 400px, 768px" src="/images/08e1653e.png" width="640"/>
</Frame>

Then take action on all these issues at once using the action bar at the bottom of the page.

<Frame>
<img alt="" class="block" data-testid="zoom-image" fetchpriority="high" height="129" sizes="(max-width: 640px) 400px, 768px" src="/images/e7ec007a.png" width="728"/>
</Frame>

Actions include:

* Change status
* Set assignee
* Set team
* Set tag
* Delete
* Snooze
* Change customer portal visibility
* Follow
* Bulk reply
* Merge
* Group issues
