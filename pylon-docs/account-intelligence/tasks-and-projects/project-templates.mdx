Create templates for sets of tasks and milestones that make it easy to track repeatable work.

Visit the [Project Templates](https://app.usepylon.com/settings/project-templates) page to create a new project template or modify your existing templates.

<Frame>
<img alt="" class="block" data-testid="zoom-image" fetchpriority="high" height="1828" sizes="(max-width: 640px) 400px, 768px" src="/images/f32b1777.png" width="2346"/>
</Frame>

Project templates contain:

* Milestones
* Tasks within milestones
* Relative due dates
* Templated task owners

## Creating Projects from Templates

Projects templates can be used in [Triggers](/pylon-docs/platform/triggers) to spin up projects automatically based on any conditions.
